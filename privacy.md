# Privacy Policy

Last updated: January 2026

## 1. Introduction
SzigiBot is a Discord bot designed to provide automated server protection using a honeypot-based security system.

## 2. Data Collection
SzigiBot collects only the minimum data required to operate its functionality.

The following data may be processed:
- Discord User IDs (for moderation actions)
- Discord Server IDs and names
- Message content sent in the designated honeypot channel only
- Timestamps related to moderation events

SzigiBot does **not** read or store private messages.

## 3. Data Usage
Collected data is used exclusively for:
- Automated moderation actions
- Security logging
- Abuse prevention

Data is **never sold, shared, or used for advertising**.

## 4. Data Storage
Log data may be stored temporarily or permanently for security and auditing purposes.
Only the bot owner has access to stored logs.

## 5. Third Parties
SzigiBot does not share data with third parties.
The bot operates solely within Discord's infrastructure.

## 6. User Rights
By using a server where SzigiBot is present, users consent to this policy.
Server administrators are responsible for informing members of the bot's presence.

## 7. Changes
This Privacy Policy may be updated at any time without prior notice.

## 8. Contact
For questions regarding this policy, please contact the bot owner via GitHub.
